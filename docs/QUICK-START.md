# Quick Start Guide

Get a demo running in 5 minutes.

## 🚀 5-Minute Setup

### 1. Download (1 min)
- Go to [Verticals Index](../verticals/VERTICALS-INDEX.md)
- Click "One Seguro" (recommended for first-timers)
- Download Portuguese version

### 2. Import (2 min)
- Open Cognigy → Settings → Packages
- Click "Import Package"
- Drag and drop the `.zip` file
- Click "Import"

### 3. Assign (1 min)
- Go to Agents
- Select an agent
- Click "Flows"
- Add the imported flow
- Save

### 4. Test (1 min)
- Click agent name
- Click "Preview"
- Type: "Hi"
- Agent responds

**Done!** 🎉

---

## 🎬 Quick Demo (10 min)

### Setup
1. Open Cognigy preview
2. Load the flow you just imported

### Run Demo
You: "I want to report a claim"
Agent: "What type of claim?"
You: "Car accident"
Agent: "When did it happen?"
You: "Yesterday"
Agent: "Any injuries?"
You: "No, I'm fine"
Agent: "Where did it happen?"
You: "Downtown"
Agent: "Perfect! Let me save this..."
Agent: generates claim number

### Show Benefits
- Natural conversation
- No forms to fill
- Instant confirmation
- Available 24/7
