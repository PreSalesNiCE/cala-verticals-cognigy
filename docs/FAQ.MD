# Frequently Asked Questions

## General Questions

**Q: What are verticals?**
A: Pre-built AI Agent demonstrations organized by industry. Each includes flows, documentation, and demo materials.

**Q: Who should use verticals?**
A: Solution engineers, pre-sales, implementations teams, and anyone demoing Cognigy AI.

**Q: Do I need coding skills?**
A: No. Verticals use Cognigy's visual builder. No code required.

**Q: Can I customize verticals?**
A: Yes. They're templates you can modify, extend, and adapt.

---

## Technical Questions

**Q: What Cognigy version do I need?**
A: Version 5.0 or higher. Check your instance.

**Q: How do I install a vertical?**
A: Download the `.zip`, go to Settings → Packages → Import. [Full guide](./INSTALLATION-GUIDE.md)

**Q: Can I use multiple verticals?**
A: Yes. Each has its own package. Import separately.

**Q: Do verticals require integration setup?**
A: Most include placeholder integrations. You can skip them for demos or connect your systems.

**Q: How much storage do verticals need?**
A: Typically 50-100 MB per vertical.

**Q: Can I use verticals offline?**
A: No. Cognigy requires internet connection.

## Language Questions

**Q: What languages are supported?**
A: Portuguese (Brazil), Spanish. More coming.

**Q: Can I translate a vertical?**
A: Yes. You can modify flows and add new languages.

**Q: Does voice support accents?**
A: Voice works best with clear speech. Accents are supported but may vary.

## Demo Questions

**Q: How long is a typical demo?**
A: 10-20 minutes depending on depth. See demo script.

**Q: What should I practice?**
A: Flow transitions, key talking points, and Q&A scenarios.

**Q: What if something fails during demo?**
A: Have backup screenshots and conversation examples ready.

**Q: Can I customize the demo script?**
A: Yes. Adapt it to your audience and use case.

## Customization Questions

**Q: How do I change the company name?**
A: Edit the `DEFAULT_COMPANY` variable in script nodes.

**Q: Can I add my own integrations?**
A: Yes. Connect your APIs to the flow.

**Q: How do I add new conversation flows?**
A: Build in Cognigy, test, then export as new package.

**Q: Can I remove flows I don't need?**
A: Yes. Delete nodes or entire flows in Cognigy.

---

## Troubleshooting Questions

**Q: Import button not showing?**
A: Verify admin permissions. Clear cache. Try different browser.

**Q: Flow won't open?**
A: Assign to agent first. Refresh Cognigy. Check dependencies.

**Q: Conversations don't work?**
A: Test in Preview mode. Check integrations. Review error logs.

**Q: Mobile version not responsive?**
A: Check browser zoom. Verify XApp settings. Clear cache.
