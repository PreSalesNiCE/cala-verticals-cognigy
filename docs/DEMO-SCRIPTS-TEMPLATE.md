## Backup Plans

### If Voice Demo Fails
- Switch to web demo
- Show pre-recorded video
- Display conversation logs
- Script: "Same technology, showing you the backend"

### If XApp Won't Load
- Use screenshot walkthrough
- Narrate through each step
- Still impactful

### If Slides Don't Display
- Have printed copies
- Use speaker notes
- Tell the story verbally

### If No Internet
- Use pre-recorded demo
- Focus on conversation examples
- Deep dive into one feature
