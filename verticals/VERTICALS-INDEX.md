## ❓ FAQ

**Q: Can I customize a vertical?**
A: Yes! All verticals are templates. You can modify flows, add integrations, adjust language, etc.

**Q: What if my industry isn't here?**
A: [create a new vertical](../CONTRIBUTING.md#adding-a-new-vertical).

**Q: Can I use multiple verticals?**
A: Yes, each has its own package. You can import multiple into Cognigy.

**Q: Do I need coding skills?**
A: No. Verticals use Cognigy's visual builder. Customization doesn't require code.
