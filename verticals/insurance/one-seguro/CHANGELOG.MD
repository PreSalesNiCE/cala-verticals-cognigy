## [1.0.0] - 2025-05-04

### Added
- Initial release of One Seguro AI Agent
- Voice claims reporting in Portuguese (BR), Spanish
- Web-based claims form with XApps technology
- Multi-language support (pt-BR, es)
- Document upload capability (photos, PDFs)
- Schedule a service via Xapps and voice
- Intelligent escalation to human agents
- Email and SMS notifications
- Mobile-responsive web interface

### Features Included
- Claim type detection (accident, theft, damage, medical)
- Schedule a service
- Contextual follow-up questions
- Third-party information capture
- Form validation and error handling
- Conversation flow documentation
- Demo scripts for presentations
- Screenshots and user guides

### Documentation
- Comprehensive README
- Conversation flow examples
- Demo presentation guide
- Installation instructions

### Supported
- Voice: Portuguese, Spanish
- Web: Portuguese, Spanish
- Devices: Mobile, tablet, desktop
- Browsers: Chrome, Firefox, Safari, Edge
