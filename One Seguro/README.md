# One Seguros – AI Agent Demo

## Overview

This demo showcases AI-driven customer interactions for the insurance industry.

It is designed for pre-sales demonstrations, highlighting how conversational AI can streamline customer service and operational workflows.

---

## Use Cases

### 1. Open a Claim (Sinistro)

Customers can report an insurance claim through:
- Voice interaction
- Webchat / XApps

**Key capabilities:**
- Guided claim submission
- Data collection (policy, incident details)
- Structured flow for fast processing

---

### 2. Schedule a Service

Customers can schedule services such as:
- Locksmith
- Electrician
- General home services.

**Key capabilities:**
- Date and time selection
- Service type selection
- Confirmation flow

---

## Demo Notes

- Built entirely using Cognigy flows
- No external integrations required
- Optimized for demonstration purposes (not production)

---

## How to Use

1. Import the package (see main README)
2. Assign the flow to an existing AI Agent
3. Start the conversation using:
   - Voice (PWA)
   - Chat interface
